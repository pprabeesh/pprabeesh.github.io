Recent Success Stories

RE-ENGINEERING OF A LEGACY ENTERPRISE PORTAL

A web-based application used by the Merchandizing team of the retailer to create Line Plans was implemented using Microsoft (MS) Silverlight and supporting technologies. MS published end of support timelines across various browsers thus necessitating re-engineering of the app. Objective of the project was to identify and utilize a newer technology stack to rebuild and deploy the application with minimal impact to the users. After a detailed analysis of the legacy application and available technology options, the application was re-engineered with React-Redux stack for better performance and usability.

ENTERPRISE PURCHASE REQUISITION

The scope of the program included the design and development of a purchase requisitioning approval system for the client. The program involved building a cross platform solution for a purchase requisitioning system within the enterprise. The application was developed as a responsive single page application with Angular. The solution was also delivered as per the stringent UX design guidelines with key considerations like accessibility and multi-language support.

HYBRID MOBILE SOLUTION FOR TRAVEL

Designed and developed a hybrid mobile solution using Angular for a manufacturing client for helping the customers find best routes to destinations. The solution integrated multiple services from multiple vendors to provide a wholesome picture to the user. The application was designed as a single page application which was wrapped with the help of Cordova to create platform specific builds. The integration included working with multiple vendors including Nokia maps, various parking service providers providing real-time information and real-time public transport information.

WEB SOLUTION FOR IN-STORE RETAIL EMPLOYEES

A web solution was designed and developed for a retail client to provide an in-store web solution available across multiple devices for better customer engagement. The solution was developed as a single page application with integration to RESTFUL JSON services. The in-store web application was used by store associates for providing additional information and details about products to customers. This was also converted into a hybrid mobile solution with scanning and payment features for mobile devices like tablets and smartphones.

WEB SOLUTION FOR MFG INSIGHTS

Designed and developed a web solution using AngularJS for a manufacturing client which helped in improved tracking of the manufacturing process for the client. The web solution was built as a single page application using AngularJS having integration with in-house services for manufacturing insights. The solution also included the integration of charts using D3.js. The solution was also made responsive using bootstrap framework and worked across browsers and platforms including old IE versions.
